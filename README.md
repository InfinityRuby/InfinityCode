**無限** 代表著我們的無限可能性，以及我們身為新手工程師的滿腔熱血。

**解碼戰** 則是期許自己能在每次遇到難題時，努力去跨越，並且戰勝它！

#### 貼心提示功能，讓你不再卡關卡到天荒地老

#### 資訊互通有無，提問發表交流樣樣通

#### 覺得害羞嗎？匿名發文讓你勇敢發文

#### 排名及積分成績表，激發使用者的好強心
